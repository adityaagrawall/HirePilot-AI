## 💡 What is HirePilot AI?!

**HirePilot AI** is an intelligent, full-stack resume optimization platform designed to bridge the gap between job seekers and Applicant Tracking Systems (ATS). 

In today's job market, over 75% of resumes are filtered out by automated screeners before reaching a human recruiter. HirePilot AI solves this problem by using advanced Large Language Models (LLMs) to analyze resumes against specific job descriptions, identify key skill gaps, rewrite descriptions using high-impact metric-driven bullet points, and instantly build a perfectly formatted, print-ready single-page resume.

---

## 🌟 Key Features

### 📊 1. Instant ATS & Match Scoring
* Get a comprehensive compatibility score showing how well your resume matches the target job description.
* Analyze strengths, weaknesses, and potential interview questions tailored to your profile.

### 🔍 2. Keyword Gap Analysis
* Automatically detects critical technical skills, methodologies, and industry-standard keywords missing from your current resume.
* Pinpoints exactly what terms you need to include to rank higher in recruiter searches.

### ✍️ 3. Metric-Driven STAR Rewrites
* Transforms passive job descriptions into active achievements using the **STAR (Situation, Task, Action, Result)** method.
* Naturally weaves in missing keywords while highlighting metrics, percentages, and business impact.

### 📄 4. World's Best Single-Page Resume Builder
* Generate a meticulously styled, ATS-compliant, single-page resume with one click.
* Custom-engineered margins, font sizes, and layout constraints to guarantee it fits perfectly on standard A4 paper without pagination errors.
* Instant PDF download leveraging native browser print styling.

### ✉️ 5. AI Cover Letter Generator
* Get a tailored, persuasive cover letter matching the tone of the job posting and highlighting your most relevant qualifications.

## 🎨 Modern Architecture & Design

### 💎 Gorgeous UI/UX
Designed with a sleek, premium developer-inspired aesthetic:
- **Glassmorphic** dashboard overlays
- Harmonic dark-mode color scheme with clean purple/indigo gradients
- Micro-animations and responsive flex layouts powered by **Framer Motion** and **Tailwind CSS**
- Accessible components structured using **Radix UI** primitives

### ⚡ Blazing Fast Fullstack Core
- **Client:** React 19 Single Page App built on Vite for sub-second page loads.
- **Auth:** Complete user identity management and secure session handling via **Clerk**.
- **Backend:** Scalable Express API running on Node.js.
- **Database:** **Supabase PostgreSQL** cloud instance managed through **Prisma ORM** for ultra-fast, structured query performance.
- **AI Core:** Built with the Google Generative AI SDK, harnessing **Gemini 2.5 Flash** for rapid, deterministic JSON extractions.

- **Saves Hours of Manual Customization:** Tailor your resume for 10+ jobs in minutes instead of spending hours rewriting bullet points manually.
